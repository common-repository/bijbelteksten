== Description ==
Maakt automatisch links van bijbelverwijzingen naar de vertaling van jouw keuze.

= Special Features =
*   Herkent Nederlandse bijbelverwijzingen!
* 	Veranderd platte tekst naar echte links. Jij kiest de bijbelvertaling en ** Bijbel Plugin voor Wordpress ** doet de rest.
*	Toont de bijbeltekst in een nieuw venster of popover, zodat bezoekers op jouw website blijven. 
*	Keuze uit de de NBG51-vertaling, English Standard Version of World Messianic Bible.
*	Het werkt gewoon!

== Upgrade Notice ==
De oude 'Bijbelteksten' plugin moet worden verwijderd, deze nieuwe versie/plugin komt daar voor in de plaats.

== Installation ==
= Vanuit Wordpress =
1.  Ga naar 'Plugins > Add New'
2.  Zoek naar 'bijbelteksten plugin'
3.  Klik op de 'Activate' knop.

= Handmatige installatie =
1.  Download de plugin.
2.  Unzip het bestand.
3.  Upload de complete `bijbel-plugin` map naar je Wordpress `/wp-content/plugins/` map.
4.	Activeer de plugin via het 'Plugins' menu in je WordPress Dashboard.  
5.  Je ziet een nieuw menu item met de naam 'Bijbel Plugin' in je Wordpress Dashboard sidebar.
